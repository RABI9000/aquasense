# Smart Irrigation System (ML + Remote Sensing)

## Overview
This project builds an intelligent irrigation scheduling system using:
- Machine Learning (Random Forest)
- Weather data (NASA)
- Soil moisture modeling
- SAR satellite data (Sentinel-1)

## Features
- Baseline irrigation model
- ML-based predictive irrigation
- Uncertainty-aware scheduling
- 5-day irrigation forecasting
- SAR-based soil moisture correction (in progress)

## Structure
- src/ → core logic
- data/clean/ → processed datasets
- data/raw/sar/ → SAR images (not uploaded)

## Current Status
- ML model trained
- Smart scheduler implemented
- Forecasting working
- SAR integration in progress
